## cert verify

Verify certificate chains for hosts or certificate files

### Synopsis

Verify certificate chains for one or more targets. Targets can be
TLS endpoints (host:port) or certificate files. You can supply custom root CA
and intermediate bundles. Optionally check revocation.


```
cert verify <host:port|cert.pem> [more ...] [flags]
```
