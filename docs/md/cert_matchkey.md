## cert matchkey

Check whether a certificate and private key match

### Synopsis

Load a certificate and a private key and determine whether they
correspond to each other. Returns non-zero on mismatch unless --verbose is set
and a match is found.

```
cert matchkey -c cert.pem -k key.pem [flags]
```
